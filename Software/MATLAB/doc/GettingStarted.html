Getting Started with MATLAB Interface for Avro

Description

Provides native MATLAB support for reading and writing Apache Avro files

System Requirements

MATLAB 2017b or greater.

Features

Provides easy to use MATLAB functions to read and write Avro files
Supported features include:
  • Writing MATLAB datatypes to Avro
  • Compression supported natively for writing
  • Automatic schema generation on write
  • Reading Avro files to native MATLAB types
  • MATLAB datatypes preserved on read. i.e. write a timetable to Avro file, reads back a timetable
  • Windows, Linux and Mac compatible

Help

Get help by typing a phrase with the name avro in the help search field, or in the command window:
help avroread
or
doc avroread

Examples

To get started with using the reader and writer, open the avroexamples.mlx script.
edit('avroexamples.mlx')
